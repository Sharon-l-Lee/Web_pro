●조건문
	제어문이 실행 완료되는 경우 제어문 블록을 빠져나와 정상 흐름으로 돌아가게 된다.
	조건식의 결과에 따라 블록의 실행여부가 결정된다
	if문, switch문

●if문
	if(조건식1){
		조건식 1이 참인 경우 실행할 명령문;
	}else if(조건식2){
		조건식 1이 거짓이고 조건식 2가 참일 때 실행할 명령문;
	}else{
		조건식 1과 2가 전부 참이 아닐 때 실행할 명령문;
	}


●switch문 :if~else if보다 가독성이 좋은 경우가 있다
	switch(정수형 또는 문자형 변수/ 자바 1.7버전 이상은 문자열도 가능){
		case 변수값1 : 위의 변수가 변수값1일 때 처리할 명령문; 
			break;
		case 변수값2 : 위의 변수가 변수값1일 때 처리할 명령문; 
			break;
		.
		.
		.
		default : 위의 변수값들이 아닐 때 처리할 명령문; 
			break;
	}
