Web이란?
	WWW(world wide web)의 줄임말로 전 세계의 컴퓨터를 연결해 정보를 공유하는 데 목적이 있다.
	1개 이상의 사이트가 연결되어 있는 인터넷 서비스의 한 가지 형태이다.
		인터넷이란, 네트워크가 연결되어 있는 형태로 인터넷 안에는 많은 서비스(기능)가 존재한다.

	IP : pc가 가지고 있는 유일한 주소
	프로토콜(Protocol) : 프로토콜마다 통신 규약이 다르다
	Http : HyperText Transfer Protocol 정보를 가져와 화면상에서 봄
	FTP : File Transfer Protocol
	SMTP, POP : 메일 송수신 protocol
	DHCP : Dynamic Host Configuration Protocol 다이나믹하게 계속 사설 ip 분배해주는 하여 Web을 사용하도록 하는 protocol
	DNS : IP 주소와 도메인 주소를 이어주는 환경/시스템
	Port : 0~4자리. 프로그램번호
	EX ) http(ip)://www.naver.com(도메인):80(포트번호)/index.html

HTML이란?
	Hyper Text Markup Language의 약자. 
	웹문서를 기술하는 언어로 웹문서를 표현하기 위해 태그들로 구성되어 있다.
CSS란?
	Cascading Style Sheets의 약자
	HTML문서에 디자인적인 요소를 담아 정보 전달을 좀더 효율적으로 하기 위해 만들어진 문서.
	HTML은 정보를 가지고 있고, CSS는 화면상의 레이아웃등을 컨트롤하는 문서에 가깝다.

JavaScript (동적인 내용) -> jQuery
cf) Ajax는 웹 페이지의 특정 부분(특정 태그 부분)를 업데이트하고 서버와 데이터를 교환하는 기술

기본적인 웹 표준 문서의 상호관계
	웹표준은 웹문서의 구조와 표현(디자인) 그리고 동작을 별도로 구분한다.
	(1)HTML 웹 문서의 구조, 정보
	(2)CSS : 표현(디자인) 담당
	(3)Java Script : 동작을 담당

	이렇게 3가지 요소가 유기적으로 결합하면 웹 문서가 가벼워지고 유지보수를 하기 쉬워진다.
