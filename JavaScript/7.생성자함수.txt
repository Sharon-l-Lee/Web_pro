생성자란
	객체를 생성할 때 호출되는 단위 메소드
	function 생성자함수명(속성명들) {
           	 ...
    	   

    	   var 객체명 = new 생성자함수명(속성값들); 
	이와 같이 new 키워드로 객체를 생성하므로 생성자함수로 만든 객체를 객체 또는 인스턴스라 부른다. 
	해당 객체가 어떠한 생성자 함수로 생성됐는지 확인할 때 instanceof 키워드를 사용할 수 있다

프로토 타입
	생성자 함수로 생성된 객체가 공통으로 가지는 공간
	객체를 생성할 때 마다 동일한 함수를 계속 생성하는 경우, 이러한 문제를 해결하기 위해 프로토 타입을 이용한다.
	->동일한 메소드를 모두 프로토 타입에 넣어서 처리한다.

new키워드
	new 키워드로 함수를 호출하면 객체를 위한 공간을 만들고 this 키워드는 해당 공간을 의미한다. 
	그렇지 않은 경우, 일반적으로 this 키워드를 사용하면 window 객체를 나타내며 Window 객체에 속성을 추가한 것이 된다.
