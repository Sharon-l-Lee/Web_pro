
●반복문 : 제어문을 반복해서 실행한다(루핑)

●for문 : 반복 횟수가 정해져 있을 때 사용

	for( 초기값 ; 반복할 조건 ; 증감식 ) {
		반복할 명령문
	} 

*블록 내에서 break를 사용하면 switch문에서와 동일하게 반복문의 블럭을 빠져나가게 할 수 있다.

●while문 : 반복횟수가 정해져 있지 않고, 일정 조건을 만족할 때 까지 수행할때 사용
	while(조건식) {
		조건식이 참일 때 계속 실행할 명령문
	}

●do-while문 : 한 번은 수행 한다. 이후에는 조건이 만족되면 수행, 만족되지 않으면 수행하지 않는다.
	do{
		한번은 무조건 실행. 두번째부터는 조건식이 참이면 계속 수행할 명령문
	} while(조건식);
